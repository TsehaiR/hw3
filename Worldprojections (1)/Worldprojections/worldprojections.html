In this project I learned how to display images in different projections

Displaying the map in different projections using QGIS

QGIS allows the user to change the projection of a map by clicking the CRS button in the bottem right hand corner. From there, I was able to search for different projections by typing the EPSG number into the "filter" search bar. After selecting the desired projection and clicking "OK" the map will appear in the proper projection on the main page.
